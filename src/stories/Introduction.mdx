# netu-react-dsf-components

An implementation of the [GOV-CY Design System](https://gov-cy.github.io/govcy-design-system-docs/) in [React](https://reactjs.org)

## Usage

```sh
npm install govuk-react styled-components @types/styled-components --save
```

```jsx
import { Button } from 'govuk-react';

const MyComponent = ({ title }) => (
  <div>
    <h1>{title}</h1>
    <Button />
  </div>
);
```

See the [Storybook](https://govuk-react.github.io/govuk-react) for examples of all available components.
